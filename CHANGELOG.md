# Changelog

本文档记录 DataRoom 的重要变更

格式参考 [如何维护更新日志](https://keepachangelog.com/zh-CN/1.1.0/)
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)

## [3.1.0] - 2026-06-13

### 新增

- 新增页面历史记录删除能力
- 新增页面历史记录备注修改能力
- 新增 Spec-Kit 支持

## [3.0.0] - 2026-05-26

### 新增

- 新增设计器撤销、重做能力。
- 新增设计器返回时的未保存确认能力。
- 新增页面与大屏设计器历史记录、自动备份与回滚能力。
- 新增 Excel 数据集能力，支持基于应用库 `excel_` 表的 SQL 查询、字段元数据读取、参数解析和前端编辑器。
- 新增素材资源本地存储与 S3 可插拔存储能力，支持文件代理访问、封面更新和 Range 读取。
- 新增用户管理、访问日志和开放 API 相关能力。
- 新增前后端自动打包发布脚本。

### 变更

- 优化大屏设计器与页面设计器的操作布局、组件图层和交互体验。
- 优化组件库为弹框模式。
- 优化设计器弹框尺寸，并支持大屏标题弹框改名。
- 优化素材库封面显示效果。
- 优化组件插入流程，支持多选插入。
- 优化素材资源上传、预览、读取和删除流程。
- 优化数据库建表和初始化脚本，提升多数据库兼容性和数据完整性。
- 将前端模块名称调整为 `dataRoomFront`。
- 升级前后端依赖版本。

### 修复

- 修复 ECharts 图表默认未配置数据集时数据不显示的问题。
- 修复登录用户名和密码未去除首尾空格的问题。
- 修复主题修改不生效的问题。
- 修复大屏组件拖拽和旋转时的闪烁问题。
- 修复首次预览时页面为空的问题。
- 修复多环境数据库配置和认证异常处理问题。
- 修复前端打包失败问题，并补充 Windows 打包命令支持。
