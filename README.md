🔥DataRoom是一款基于SpringBoot3.x、JDK17、Vue3.x、Vite8.x、Element-plus、Echarts等技术栈的大屏设计器，具备大屏设计、预览能力，支持MySQL、Oracle、PostgreSQL、SQLServer、JSON、Excel文件、HTTP数据集接入，使用简单，完全免费，代码开源。<br/>

## 特别说明

❗️❗️❗️Vue3版本功能未开发完成, 请勿用于生产环境, 持续更新中、生产环境可使用`2.1.0.RELEASE`分支
