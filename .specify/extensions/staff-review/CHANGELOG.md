# Changelog

All notable changes to this extension will be documented in this file.

## [1.0.0] - 2026-04-01

### Added

- Initial release of Staff Review extension
- Command: `/speckit.staff-review.run` - Staff-engineer-level code review
- Review report template with structured findings
- Severity levels: 🔴 Blocker, 🟡 Warning, 🟢 Suggestion
- Spec compliance checking against spec.md
- Test coverage assessment
- Security vulnerability detection

### Requirements

- Spec Kit: >=0.1.0
